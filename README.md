### 달력 컴포넌트 만들기

1. CRA 없이 리액트 어플리케이션 세팅하기
2. 달력 라이브러리 선정
   - https://reactdatepicker.com/
3. 달력 기능 리스트
   - [x] 시작값, 끝값 존재
   - [x] 년, 월, 달, 시, 분, 초 설정 가능
   - [x] 인풋에 날짜값 입력하여 값 설정 가능
   - [x] 특정 구간의 값을 설정하는 기능 버튼 존재 (ex. 최근 일주일, 최근 한달)
   - [x] (유효성 검사) 시작값이 끝값보다 뒤에 입력시 에러 메세지 출력
   - [x] (디자인) 피그마 달력 디자인 적용
